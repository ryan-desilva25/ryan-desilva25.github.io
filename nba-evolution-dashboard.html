The Evolution of the NBA
A 20-Year Data Study

Investigating multi-million-row transactional shot logs, changing physical player archetypes, and league globalisation patterns using the 5DS storytelling framework.

Live Interactive Dashboard Platform

Interact directly with the live serverless application deployment running below. Use the interface tabs to isolate shifting shot location densities, inspect changing height-weight clusters, or track international talent growth via interactive choropleths.

Visualisation Architecture (Five Design Sheets Process)

The system interface balances advanced exploratory data depth for professional scouts with clean visual storytelling layouts for sports analysts, structured around the Five Design Sheets (5DS) Framework:

  • 1. Discover: Explored 20 distinct visual idioms to map out patterns tracking shot density (spatial loc coordinates), age distributions, and international draft trends.
  • 2. Define: Established operational criteria for era bucketing and position groupings to isolate macro movements from background player variance noise.
  • 3. Design: Tested alternative grid layouts (Magazine vs. Interactive Dashboard formats), prioritizing sidebars to optimize cognitive load.
  • 4. Develop: Deployed a cohesive interface using the 'Superhero' dark aesthetic, transparent chart layer configurations, and highly interactive Plotly/Leaflet render objects.
  • 5. Deliver: Created an optimized serverless asset delivery strategy powered by data pre-aggregation to maximize fast UI response.

Data Pipeline & Engineering Execution

Translating multiple dense data assets into a highly performant application layer required explicit data restructuring steps:

  • Ingestion & Schema Standardization: Consolidated 17 individual season shot-log CSV files into a unified dataset using janitor::clean_names().
  • Missing Value Imputation: Successfully resolved thousands of missing positional coordinates across season boundaries by performing categorical left-joins on player metadata keys.
  • Geospatial Standardization: Unified international player origin naming conventions to align perfectly with Natural Earth map shapefiles, eliminating discrepancies across 15+ tracking regions.
  • Performance Optimization: Engineered an automated ETL routine that pre-calculates analytical data matrices (e.g., historical True Shooting percentages, usage statistics, and geometric counts), reducing application load times down to sub-second responses.
